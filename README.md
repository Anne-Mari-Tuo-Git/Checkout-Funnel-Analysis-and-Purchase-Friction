# Checkout-Funnel-Analysis-Purchase-Friction
Google Merchandise Store Analysis 📊 E-commerce study using GA4 and Power BI. I identified a 49% checkout friction point and high drop-off rates (>80%) across categories. Using DAX for Revenue and AOV, the project highlights Apparel as the top volume driver and Campus Collection as the efficiency leader with a 30.2% conversion rate. 🚀
